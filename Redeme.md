## 📋 Deskripsi

NEXUS SCHOOL adalah sistem manajemen sekolah dengan tema cyberpunk neon yang terintegrasi dengan **Supabase** sebagai backend. Aplikasi ini menyediakan fitur lengkap untuk mengelola data sekolah termasuk siswa, guru, kelas, jadwal, nilai, absensi, kas kelas, pengumuman, dan chat real-time.

## ✨ Fitur Utama

- 🔐 **Autentikasi** - Login/Register dengan Supabase Auth
- 🎓 **Manajemen Siswa** - CRUD data siswa
- 🧑‍🏫 **Manajemen Guru** - CRUD data guru
- 🏫 **Manajemen Kelas** - CRUD data kelas
- 📊 **Nilai** - Input dan lihat nilai siswa
- ✅ **Absensi** - Catat kehadiran siswa
- 💰 **Kas Kelas** - Transaksi keuangan kelas
- 📢 **Pengumuman** - Posting pengumuman
- 🗂️ **File Management** - Upload dan kelola file
- 💬 **Chat Real-time** - Chat kelas dengan Supabase Realtime
- 📅 **Kalender** - Lihat agenda dan pengumuman
- 🛡️ **Role-based Access** - Admin, Ketua Kelas, Siswa
- 🌓 **Dark/Light Mode** - Tema dengan efek neon
- ✨ **Animasi Partikel** - Efek latar belakang interaktif

## 🚀 Teknologi

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Supabase (PostgreSQL, Auth, Storage, Realtime)
- **Library**: Chart.js, Orbitron Font, Share Tech Mono
- **Styling**: CSS Custom Properties, Glassmorphism
